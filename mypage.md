# Mypage(마이페이지)


**문제 제기**
- 사용자의 정보를 확인하고 싶다.

**기능 요약**
- 학생이 결제한 수강 목록 및 상세 내용 확인
- 학생이 관심 기능을 추가한 강의 목록 확인
- 강사의 개설완료된 강의 목록 및 상세 내용 확인
- 사용자가 게시판에 작성한 질문을 확인

**기능 설명**
- 학생이 결제한 수강 목록 및 상세 내용 확인

- 강사의 개설완료된 강의 목록 및 상세 내용 확인
- 사용자가 게시판에 작성한 질문을 확인
